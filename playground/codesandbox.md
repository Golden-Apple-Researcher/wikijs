# Codesandbox

Codesandbox ist nicht nur ein VS Code Fork der im lokal Browser laeuft, sondern stellt zusaetzlich eine VM bereit auf der das backend laeuft, versioniert die Projekte automatisch mit git und stellt eine einfach Share Funktion bereit, ueber die sie sich vielfaeltig teilen bzw. zeigen lassen.
